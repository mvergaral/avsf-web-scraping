# Guía de comando Bash

```console
# Para revisar los documentos y carpetas dentro de una ubicación, 
# reemplazar [carpeta] por el directorio a consultar, si se deja vacio se consultara el directorio actual

ls [carpeta]

# Para acceder a una carpeta dentro de los directorios, 
# reemplazar [carpeta] por el directorio al que se quiere acceder

cd [carpeta]

# Para volver a la carpeta anterior, desde dos puntos se accede al anterior, 
# agregando más puntos más atras se vuelve entre los directorios

cd ..

# Para volver a  la carpeta Root

cd

# Para crear una carpeta, reemplazar [carpeta] por el nombre de la carpeta a crear

mkdir [carpeta]

# Para crear un archivo, reemplazar [archivo] por el nombre del archivo a crear

touch [archivo]

# Si se tiene instalado visual studio code 
# junto a la extensión code (https://code.visualstudio.com/docs/setup/mac#_launching-from-the-command-line)
# Para abrir un archivo, reemplazar [archivo] por el archivo que se quiere abrir en vscode. 
# Tambien se puede reemplazar por una carpeta para abrir todo un directorio como entorno de trabajo,
# si se reemplaza por un punto (.) se abre el directorio actual

code [archivo]

# Para ejecutar un codigo en python,
# reemplazar [archivo] por el archivo que se quiere ejecutar

python [archivo]
```
